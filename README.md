# RidingRecord

## Overview
This is an web application for making, reviewing and managing horse riding records.

## Features
- Addtion of horse riding record to record DB
- Display past records
- Delete a certain record

## Tech Stack
- Python
- FastAPI
- SQLite
- HTML
- JavaScript

## Setup

```bash
python -m venv .venv
.\.venv\Scripts\activate
python -m pip install -r requirements.txt
```
## Run
```bash
python -m uvicorn main:app --reload
```

## Open
```browser
http://127.0.1:8000
```
